Bem Vindo ao Tópico 01 - Conhecendo o Excel

A interface inicial do Excel consiste em ferramentas de trabalho separadas por Guias,
como é mostrado na imagem a baixo ↓


ARQUIVO: O guia arquivo serve para salvar ou criar novos arquivos no Excel

PAGINA INICIAL: O guia página inicial é a interface principal do Excel
onde será mostrado diversas formatações como: estilos, fonte, alinhamento, cores etc..

INSERIR: o guia inserir é responsável por inserir elementos no Excel
seja ele interno ou externo, como formas geométricas,gráficos, desenhos ou imagens do google

LAYOUT DA PÁGINA: o guia layout é o local onde é configurado os ajustes para a impressão do arquivo se necessário

FÓRMULAS: o guia fórmulas disponibilizará suporte para uso de fórmulas no Excel

quando falamos de Excel a nivel básico, esses guias serão os mais utilizados

como podemos ver na imagem acima, temos um L10 escrito na lateral esquerda no meio da imagem
este L10 mostra a célula presente, lembrando que no Excel, uma célula é formada pelo encontro de uma linha com uma coluna que será explicado no tópico 02

ESPAÇO DA FÓRMULA: ao lado tem um espaço em branco com um Fx ao lado, neste espaço será mostrado as funções/fórmulas que estão sendo usadas no Excel

CRIAR NOVA ABA: no canto inferior esquerdo temos a opção + onde podemos criar uma nova aba dentro da mesma planilha
permitindo assim uma organização de arquivos dentro de um único projeto
